# DiffusionAAE
Implementation code of DiffusionAAE.

# Data Preparation
Put your datasets(IndianPines,Salinas, PaviaUniversity) in "datasets" directory (.mat files).

# Usage

Set your configs in main.py and run it.

```
python main.py
```
